Administration Tab

The Administration tab enables users with the appropriate permissions to easily perform database administration tasks.

Database Admin

The default Administration view is the Database Admin view.

This view displays the status of the database and enables you to stop or restart Greenplum Database in any of the following modes:

  • Smart (default). Smart shut down. If there are active connections, this command fails with a warning.
  • Fast. Fast shut down. Any transactions in progress are interrupted and rolled back.
  • Immediate. Immediate shut down. Any transactions in progress are aborted. This shutdown mode is not recommended. This mode kills all postgres processes without allowing the database server to complete transaction processing or clean up any temporary or in-process work files.
