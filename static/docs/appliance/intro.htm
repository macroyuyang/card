About Pivotal Greenplum Command Center

The Pivotal Greenplum Command Center is a web-based management console that provides monitoring and administration for your Pivotal Greenplum Database.

Pivotal Greenplum Command Center manages:

On the Data Computing Appliance (DCA):

  • Greenplum Database Module (4.2.6.3 or higher; 4.2.7.1 is recommended for full functionality of this release)
  • DIA (Data Integration Accelerator) Module
  • Greenplum Data Computing Appliance (DCA) Hardware (V1.x and V2.x)

Software-only Environments:

  • Greenplum Command Center manages your Greenplum Database software-only environment (4.2.6.3 or higher; 4.2.7.1 is recommended for full functionality of this release).

Pivotal Greenplum Command Center provides the following functionality:

  • Administrative Controls

    • Ability to start/stop the database
    • Ability to recover / rebalance segments

  • An Interactive View of System Metrics

    • Realtime
    • Historic (configurable by time)

  • *An Interactive View of System Health

    • Hardware health
    • Software health 

    * Health monitoring is only available for Greenplum Command Center running on a Data Computing Appliance (DCA)

  • Query Monitoring

    • Ability to view, search, prioritize, or cancel any query in the system.
    • Ability to views the internals of any query, including the query plan, query plan iterator-level details, and real-time information about individual scans and joins.

  • Workload Management

    • Ability to configure resource queues
    • Ability to prioritize users
