Exporting Data

You can export the data from any graph or table on any Command Center screen to a CSV (comma-separated values) file.

For example, to export the data in one or more of the graphs on the Dashboard, click the Export button in the upper right corner of the Dashboard.

An Export dialog appears. This dialog lists all the graphs on the Dashboard (Queries/Jobs, CPU, and Memory). Check the box next to the graph(s) you want to export, or click All to export the data from all the graphs, then click OK.

A Save As dialog appears with a default name of the first graph type appended with the current time/date stamp, and a .csv extension.

Keep or edit the file name and location, then click Save to save the file as a .cvs spreadsheet file.

If you are on a Command Center screen where there is only one graph or table to export, for example the Query Monitor screen, once you click the Export button, you are directed to the Save As screen immediately.
