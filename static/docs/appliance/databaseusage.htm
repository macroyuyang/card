Administration Tab

Database Usage Report

The Database Usage Report provides you with a summary of your database storage use.

In addition to providing summary storage information, you can also view the summary storage information categorized into different storage types: Heap, Append Only, and Column Oriented.

Below the summary information, you can view storage information for each individual database. Note that there will always be a Command Center database displayed here.

This page is empty until you generate an initial report (Generate Report). Subsequent views of this pages displays the last report run, along with the data and time when it was run.

The time it takes to generate this report depend upon the size of your cluster. Generating this report should not affect system performance.
