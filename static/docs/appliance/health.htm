Health Tab

Important: The Health tab is only displayed if you are running Pivotal Greenplum Command Center on an EMC Data Computing Appliance (DCA).

The Health tab is a simple and interactive way to view the health of your Greenplum Database clusters.

You can export the data from any graph or table in the Command Center UI by clicking the Export icon in the upper right. See "Exporting Data" for more details.

The Health tab displays Data Computing Appliance-specific information about the status of your system, and shows collected SNMP data for the following items:

  • Greenplum Database Hardware Health

    • GPDB Masters Hosts
    • GPDB Segments Hosts
    • Admin Switch
    • DIA (Data Integration Accelerator) Host (commonly used for ETL hosts) 
    • Interconnect Switch

  • Greenplum Database Software Health

    • Database

The Health tab displays several consolidated views of the health of these components as well as a detailed view of a selected component.

Consolidated Views

The Consolidated Views display of the health of the system as a whole. Each of the consolidated views uses charts and tables to display the system health of both your Database clusters.

The current status of these EMC DCA components is indicated by:

  • Normal (Green)
  • Warning (Yellow)
  • Error (Red)
  • Unknown (Gray)
  • Unreachable (Black)

The Unknown status does not indicate that the Pivotal Greenplum Command Center was unable to determine a status for a component. Unknown is a valid status that is returned by an SNMP agent when it successfully communicates with a component, but cannot determine the state of the component.

Click on any component in any of the consolidated views to see more detailed information in the Details Area.

Health by Status

This pie chart shows the health of your system by status.

Click on any segment of the pie chart to populate the other views with only components that are in that state (normal, error, and so on.)

Health by System Component

The other two consolidated views show the number of hosts/switches, their names, and their current status as follows.

Hover over any component in the bar chart to display its name and current status.

Note that this view is filtered by status (see Health by Status). Click Summary to remove the filter and display all components.

Health Details

The page list host components and associated SNMP information received by Greenplum Command Center.

 

This information includes:

  • Component Name
  • Status
  • Message

You can filter the items that display on this page by their current status. Select or clear the status check boxes to filter the list as required. For example, to view only items with a status of Error and Unreachable, select these two checkboxes and clear the Normal, Unknown, and Warning checkboxes.
