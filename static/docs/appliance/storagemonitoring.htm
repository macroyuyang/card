Storage Monitoring

The Storage Monitoring page lets you view storage metrics over time.

This page provides you with a convenient way to view Disk Usage information for Greenplum Segments, Masters, and Hadoop hosts at the same time.

This page provides you with a convenient way to view Disk Usage information for Greenplum Segments and Master hosts at the same time.

You can export the data from any graph or table in the Command Center UI by clicking the Export icon in the upper right. See "Exporting Data" for more details.

You can see how much disk space has been used/is free in the Disk Use Summary pane. This data is displayed in realtime, as shown below.

Hover over any component's disk use summary to display more information, for example

Click any host in the Disk Use Summary section to populate the other sections of this screen with more granular storage information about that host.

Click on any host, and the table below is populated with realtime information about all individual hosts of that type.

Click on any host, and the graph on the right is populated with historical information about all individual hosts of that type.  For example, click on GP Segments and you can see the total GP Segment disk usage over time in the graph on the right.

Use the drop-down menu on the disk usage over time graph to specify the period of time for which you want to see disk usage, one of either: 1 hour, 6 hours, 24 hours, 7 days, 2 weeks, or 3 months. Select the time period most appropriate for your system.
